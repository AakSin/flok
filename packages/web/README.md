# flok-web

Web server for Flok.

See [flok](https://github.com/munshkr/flok) for more.

*Work in progress*


## Install

```
npm install -g flok-web
```


## Usage

To start a local server/hub, just run on your terminal:

```
flok-web
```

Run `flok-web --help` to see all available options.
